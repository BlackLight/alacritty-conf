# alacritty-conf

Maddison's [Alacritty](https://github.com/alacritty/alacritty) configuration!

#### Prerequisites

- GNU Make
- [jq](https://github.com/stedolan/jq)
- [yq](https://github.com/mikefarah/yq)

#### Build & Install

```
$ make install
```
